## Overview

TechBros is a responsive and modern agency website built with HTML, CSS, and JavaScript. It features smooth animations, clean layouts, responsive design, and interactive user experiences.

This project demonstrates front end development skills, UI/UX principles, responsive web design, and modern JavaScript techniques.
